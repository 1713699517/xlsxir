    Xlsxir v1.5.1 Xlsxir.ConvertDate

    Converts an ISO 8601 date format serial number, in char_list format, to a date formatted in Erlang :calendar.date() type format (i.e. {year, month, day}).

    from_serial(serial)

    Receives an ISO 8601 date format serial number and returns a date formatted in Erlang :calendar.date() type format.

    Parameters

    • serial - ISO 8601 date format serial in char_list format (i.e. 4/30/75 as ‘27514’)

    Example 

    iex> Xlsxir.ConvertDate.from_serial('27514')
{1975, 4, 30}
